# 🎟️ Movie Ticket System

This is a beginner Python project that simulates a movie ticket booking system.

## 🚀 Features
- Checks user age eligibility
- Determines access to evening shows
- Applies membership discounts
- Adds extra charges for weekends and evening shows
- Calculates final ticket price

## 🧠 What I learned
- Python variables
- If/else statements
- Logical operators (and, or)

## ▶️ How to run
Run the file using Python:
python ticket_system.py# Build-a-Movie-Ticket-Booking-Calculator
